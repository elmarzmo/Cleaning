# Cleaning Service Website

## Overview
A production-ready website built for a **real cleaning services business**, designed to showcase services, collect customer inquiries, and provide a professional online presence. 
The site is deployed for public use and optimized for mobile and desktop users.

## Features
- Service listings with clear descriptions
- Responsive design optimized for mobile devices
- Server-rendered pages using Handlebars
- Backend routing for dynamic content
- Contact / inquiry handling
- Clean, user-friendly UI for real customers

## Tech Stack
- Backend: Node.js, Express
- Frontend: Handlebars, HTML, CSS, JavaScript
- Other: MVC architecture, responsive design

## What I Worked On
- Designed and implemented Express server architecture and routing
- Built dynamic Handlebars templates for reusable page components
- Structured backend logic to support service data and customer inquiries
- Implemented responsive layouts for cross-device compatibility
- Prepared the application for deployment using a cost-effective hosting solution
